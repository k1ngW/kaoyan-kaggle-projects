# 项目一：Kaggle House Prices — 回归分析实战

## 基本信息

| 项目 | 详情 |
|------|------|
| 竞赛名称 | House Prices - Advanced Regression Techniques |
| 平台 | Kaggle Getting Started |
| 链接 | https://www.kaggle.com/c/house-prices-advanced-regression-techniques |
| 状态 | 永久开放，可随时提交 |
| 任务 | 用79个特征预测房价 SalePrice |
| 评估指标 | RMSE（对数空间，即预测 log(SalePrice) 的均方根误差） |
| 参赛规模 | 14万+ 队伍 |
| 计划时间 | 2026年8月15日 — 9月30日（6周，每天≤1h） |

## 为什么选这个竞赛

1. **纯回归问题** — 直接对标432统计学「回归分析」全部考点
2. **数据干净** — 不需要爬虫，79个特征含义明确，缺失值可控
3. **资源丰富** — 14万队伍意味着无数公开 Notebook 可供学习
4. **排名可验证** — Kaggle public score 直接写在简历上，面试老师可以查
5. **难度适中** — Getting Started 级别，不会花太多时间在调参上

### Phase 2: 特征工程 (8/22-8/28) — 1周
- [ ] Target 变换：log(SalePrice) 处理正偏，验证正态性改善

- [ ] 异常值检测：Grubbs检验 / IQR方法
- [ ] **432关联**: 数据预处理、正态变换（茆诗松 Ch3）

### Phase 3: 建模对比 (8/29-9/07) — 1.5周

### Phase 4: 模型诊断 (9/08-9/18) — 1.5周
