## 3. 模型家族谱系

```
                        机器学习模型
                             │
        ┌────────────────────┼────────────────────┐
        │                    │                    │
    线性模型              树模型              神经网络
        │                    │                    │
  LogisticRegression    决策树 (基础)         MLP (基础)
  Ridge/Lasso               │                    │
                            ├── Bagging ── RandomForest
                            │                    │
                            ├── Boosting ──┐
                            │   ├── XGBoost  │ 自适应增强
                            │   ├── CatBoost │ (AdaBoost)
                            │   └── LightGBM │
                            │               │
                            └── 直方图 ─────┘
```

### 树模型演进史

```
1963  ── 决策树 (Decision Tree)               一根筋，容易背答案
1984  ── CART (分类回归树)                    加了剪枝，防止过拟合
2001  ── Random Forest (随机森林)             多棵树投票，解决"一根筋"
2001  ── Gradient Boosting (梯度提升)         一串树，每棵修正前一棵
2016  ── XGBoost                              工业级实现，Kaggle 统治
2017  ── LightGBM                             更快、更低内存
2017  ── CatBoost                             原生类别特征 + GPU
```

### 神经网络演进史

```
1958  ── Perceptron (感知机)                  单层，只能线性分割
1986  ── MLP (多层感知机 + 反向传播)           能学非线性的"万金油"
2012  ── CNN (卷积) / RNN (循环)              图像/序列专用
2017  ── Transformer / Attention              "注意力"机制诞生
2024  ── RealMLP (表格专用 MLP)               数值嵌入 + 鲁棒缩放
2025  ── TabM (表格专用 Attention)            Attention 做 tabular
```
